# Welcome to #ash

**#ash** (pronounced "hash") is your self-hosted, offline-first markdown knowledge base.

Your notes are plain `.md` files stored directly on disk — no proprietary formats, no lock-in. You own everything.

## What you can do

- **Write** — create and edit markdown notes with a live preview
- **Organize** — arrange notes in any folder structure you like
- **Search** — find anything across your entire vault instantly
- **Link** — connect notes with `[[wiki-links]]`
- **Sync** — keep your vault in sync across devices with the desktop client
- **Work offline** — the desktop client queues changes and syncs when you reconnect

## Navigation

Use the file tree on the left to browse your vault. Click any note to open it.
